# Mortar Demo

Demo project for Mortar / Flow / Dagger2.


## Showcase

- Mortar
- Flow
- Dagger2
- Screen (with Flow-path)
- ScreenScoper : `ScreenScoper`
- Subscreen (screen that is included in another screen) : `LoginScreen`
- Additional custom view injected to screen : `BannerView`
